
# MedicoCare

Developed a full-stack online hospital management system with modules for patient management. 
Modules include:




## Features

- Log in page
- Add New Patient
- Available Rooms
- Update Info
- All Doctors Info
- All Patient Info
- Ambulance
- Patient Discharge


## Technologies used
- Java Swing
- MySQL Workbench
- Abstract Window Toolkit(AWT)
- IntelliJ IDEA(IDE)
